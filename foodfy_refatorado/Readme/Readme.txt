Como usar?
Você precisa ter instalado o Node.js - https://nodejs.org/en/

Rode a aplicação no seu terminal seguindo os passos:

```
Instale as Dependências
comando: npm install
Inicie o Servidor:
comando: npm start
Acesse o Servidor no Seu Navegador
na url digite: localhost:9000
```
Para o funcionamento do servidor, siga também os seguintes passos:

1 - Remova da pasta "data" o arquivo data.js

2 - Remova da pasta "packages" os arquivos package e package-lock.

3 - Remova da pasta "server" o arquivo "server.js".

4 - Entre na pasta "node_modules", remova o arquivo "node_modules.rar" e depois extraia o conteúdo do arquivo.
